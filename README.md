# Arrythmia-Classifier-using-GNN

## Prerequisite

- pytorch (1.9.0+)
- dgl (0.8.0 / cuda 11.3)
- dgl installation = [here](https://www.dgl.ai/pages/start.html)
- networkx
- numpy
- torchmetrics

## How to execute 
1. put data into training and testing folder (```npz file``` recommend)
2. run main.py
